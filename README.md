# 🌤️ Weather App
A beautiful weather UI with glassmorphism design, city search, and 5-day forecast — built with pure HTML, CSS, and JavaScript.
## Live Demo
Open `index.html` in any browser — no installation needed.
## Features
- Search weather by city name
- Displays temperature, humidity, wind speed, feels-like
- 5-day forecast with weather icons
- Glassmorphism card with dark navy background
- Pre-loaded cities: Chennai, Mumbai, Delhi, London, Tokyo
## Built With
- HTML5
- CSS3 (Glassmorphism, Backdrop Filter)
- Vanilla JavaScript
